## Paper factsheet

The paper proposes a decision-theoretic approach to steganography in LLM reasoning. Rather than comparing a suspected signal to a benign reference distribution, it defines steganography through asymmetry in usable information between a Receiver and a Sentinel. The core measure is a steganographic gap based on generalized V-information.

Main claimed contribution:
- Generalize V-information to a utility-based measure for usable information.
- Define Receiver-usable information and Sentinel-usable information for the same signal Z, then define the steganographic gap as their difference.
- Empirically test the measure on Rot13 traces, codebook mismatch, auditing-task dependence, test-time scaling, trained paraphrasing, and public encoding schemes.

Evidence read:
- Abstract and Section 1 motivate LLM steganographic reasoning and the absence of a benign reference distribution.
- Section 3 defines generalized V-information.
- Section 4 defines the Receiver/Sentinel formalism and the steganographic gap in Equations 7-8.
- Section 6.1 validates sensitivity to Rot13 encoding probability, but gives Receiver and Sentinel different traces: Receiver observes original reasoning while Sentinel receives the possibly ciphered version.
- Section 6.5 trains a paraphraser to minimize the same gap later used for evaluation.
- Appendix F provides experiment prompts and training details.

Most decision-relevant observations:
- The conceptual pivot from distributional indistinguishability to downstream utility asymmetry is strong and relevant.
- Section 6.1 does not cleanly validate the same-signal quantity defined in Section 4, because Receiver and Sentinel are not evaluated on the same Z.
- Later experiments are useful existence proofs but are mostly controlled/synthetic and partly in-distribution with respect to the evaluation objective.

- A same-Z experiment where both Receiver and Sentinel observe exactly the same trace but differ only in decoder family, key knowledge, or computational resources.

## Master synthesis

This is a conceptually interesting paper with a real safety motivation. The decision-theoretic formulation is the strongest part: measuring whether a signal gives a Receiver more usable task information than a Sentinel is a pragmatic alternative when no benign reference distribution exists. I would accept the paper weakly because the formalism is useful and the controlled experiments demonstrate several expected properties. The score is capped because the main empirical validation in Section 6.1 violates the "same Z" comparison in the formal definition, and the mitigation experiment is partly in-distribution with respect to the optimized metric. These are significant soundness/evidence concerns but not enough to erase the conceptual contribution.
